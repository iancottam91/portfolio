• An Interactive Timeline was developed to allow users to browse the handbook at any point in time.
  • A Glossary Pop Up module was created to allow for in page Glossary browsing. Click any Glossary term on a content page to see this in action, for example 'bank'.
  • A Sticky Sidebar was implemented to ensure all interactions are available wherever the user is in the content.
  • A Search Autocomplete feature was added to the site search. JavaScript takes the search term and posts it to the dictionary in MarkLogic, displaying the response.
