• An open source tool called HTTrack was used to scrape the content from the website.
  • PHP was then used to process the scraped HTML, removing unncessary elements and adding new ones to the 1,000s of HTML files downloaded.
  • Java and JavaScript were used to index the files create a power offline search. The search allowed, boolean, keyword and title searches.
  • NWJS was then used to package the product into an application the could be run without internet connection.
